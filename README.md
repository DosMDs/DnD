# Obsidian DnD — Vault для D&D кампаний

Готовый Obsidian vault для ведения D&D кампаний с интерактивными карточками,
автоматическим созданием сущностей и календарём Харптоса.

## Возможности

- Создание кампаний, сессий, NPC, локаций, квестов, предметов, персонажей игроков, логов — одной кнопкой
- Автоматические двусторонние связи между сущностями
- Календарь Харптоса с отслеживанием игрового времени
- Интерактивные кнопки (Meta Bind) для управления статусами без ручного редактирования
- Dataview-таблицы на страницах кампании и в Home

## Установка

1. Скачать репозиторий как ZIP и распаковать
2. Открыть папку как новый vault в Obsidian
3. Установить плагины через **Settings → Community plugins → Browse**:

| Плагин | ID | Версия |
|--------|----|--------|
| Dataview | `dataview` | ≥ 0.5.68 |
| Templater | `templater-obsidian` | ≥ 2.x |
| Meta Bind | `obsidian-meta-bind-plugin` | ≥ 1.4.7 |
| JS Engine | `js-engine` | ≥ 0.3.3 |

4. Включить все четыре плагина
5. Включить CSS-сниппет **dnd-actions** в Settings → Appearance → CSS snippets

> **Важно:** бинарники плагинов (`main.js`) намеренно не включены в репозиторий.
> Их нужно устанавливать через Obsidian — это гарантирует актуальную версию.

## Структура vault

```
00_System/
  JS/              — Templater-скрипты (CommonJS)
    dnd_helpers.js — Утилиты для Templater
    create_*.js    — Скрипты создания сущностей
  MetaBindJS/      — Скрипты для кнопок Meta Bind (ESM)
    dnd_mb_helpers.js — Утилиты для Meta Bind / JS Engine
    *.js           — Действия кнопок
  Templates/System/ — Шаблоны-обёртки для Templater
  Media/           — Вложения (картинки, карты)

01_Home/
  Home.md          — Главная страница с дашбордом кампаний

02_Campaigns/      — Папка для кампаний (игнорируется в .gitignore)
  001_Campaign/
    Campaign.md
    Sessions/
    NPCs/
    Locations/
    Quests/
    Players/
    Items/
    Logs/
    Assets/
```

## Использование

Открой `01_Home/Home.md` → нажми **Создать кампанию** → заполни поля.

Внутри кампании все действия доступны через кнопки на странице `Campaign.md`.

## Примечание по архитектуре

Проект содержит два файла-хелпера:
- `dnd_helpers.js` — для Templater (CommonJS, `module.exports`)
- `dnd_mb_helpers.js` — для JS Engine / Meta Bind (ES Modules, `export`)

Это намеренное архитектурное решение: плагины Templater и JS Engine используют
несовместимые системы модулей, поэтому единый файл технически невозможен.
Математика календаря Харптоса присутствует в обоих файлах — это единственное
дублирование, которое нельзя устранить без смены одного из плагинов.
